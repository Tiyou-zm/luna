## 一、引言

### 1.1 编写目的

本文档面向软件著作权登记审核人员、平台运营管理人员及最终用户，系统说明 Luna AI - ClawSolo 微信小程序的软件功能、使用方式及业务范围。文档基于实际运行的微信小程序界面进行核查与撰写，确保所述内容与实际产品功能一致。

### 1.2 软件概述

Luna AI - ClawSolo 是一款面向自媒体创作者的智能内容生成与数据分析微信小程序。软件集成了大语言模型对话能力、多平台内容创作辅助、自媒体账号趋势数据监测、素材云存储管理、套餐计费与微信支付等核心功能，旨在为自媒体创作者提供从内容创意、创作执行到数据监测的一站式AI工作平台。

软件采用 Taro 跨端框架开发，后端依托 Supabase 云服务平台，AI 对话能力由 Hermes Agent（Luna 2.0）大语言模型提供，素材存储使用腾讯云对象存储（COS），支付环节接入微信支付能力。

### 1.3 读者对象

- 软件著作权登记审核人员
- 平台运营与管理人员
- 自媒体创作者（最终用户）

## 二、系统综述

### 2.1 设计目标

Luna AI - ClawSolo 的设计目标是为自媒体创作者提供一个轻量级、一体化的AI内容创作工作站。创作者无需在多个工具之间切换，即可在单一小程序内完成内容创意生成、素材管理、数据趋势分析和账号健康监测等核心工作。

软件强调以下设计原则：

- **一站式创作**：在一个平台内完成从创意到素材的全流程
- **多平台适配**：支持微信公众号、小红书、抖音、微博、飞书文档等多平台内容风格
- **数据驱动**：通过趋势数据看板帮助创作者了解账号表现
- **按需计费**：提供从免费版到企业版的多种套餐选择，支持算力充值

### 2.2 主要功能特点

软件具备以下主要功能特点：

1. **AI智能对话**：基于 Hermes Agent 大语言模型，支持多轮对话、文件上传分析、流式输出展示
2. **趋势数据看板**：展示近7天自媒体账号关键指标趋势，支持日、周、月多粒度切换
3. **自媒体账号绑定**：支持小红书、抖音、B站、微博、微信公众号、快手等六大平台账号绑定与监测
4. **AI账号健康诊断**：基于已绑定账号信息，由AI生成账号发展阶段判断、运营问题排查和优化建议
5. **素材云存储管理**：基于腾讯云COS实现图片、视频、音频、文档等多类型文件的云端存储与分类管理
6. **多档套餐计费**：提供6档套餐（免费版至企业版），支持微信支付购买和算力充值
7. **消耗记录追踪**：详细记录每次AI生成服务的类型、用量、扣费金额和余额变化
8. **会话历史管理**：支持多会话管理，会话列表可查看、切换和删除

### 2.3 功能架构简介

软件功能架构由以下四个层级组成：

1. **对话与创作层**：以AI对话为核心，支持内容创意生成、文件分析、快捷问题推荐
2. **数据监测层**：提供趋势数据看板、账号绑定管理、AI健康诊断
3. **资产管理层**：素材云存储、聊天历史归档、会话管理
4. **商业运营层**：套餐展示与购买、微信支付、消耗记录、订单管理

## 三、系统使用前提

### 3.1 运行环境

- **终端要求**：安装微信客户端的智能手机（iOS或Android系统）
- **微信版本**：建议微信8.0及以上版本
- **网络要求**：需连接互联网，建议使用稳定的Wi-Fi或移动数据网络

### 3.2 账号要求

用户首次使用软件时，可通过以下两种方式完成登录：

1. **微信一键登录**：授权微信登录后，系统自动获取用户openid并创建账号，无需额外注册
2. **用户名密码登录**：已有账号的用户可通过用户名和密码直接登录

登录后，用户默认获得免费版套餐权限，可进行基础AI对话和内容创作。

### 3.3 使用关注事项

- 软件内的微信支付功能仅在微信小程序正式环境中可用
- 非微信环境（如H5浏览器）中，支付功能将被禁用，但其他功能可正常使用
- AI对话服务需要消耗算力或套餐配额，免费版用户有固定的体验额度
- 素材上传功能依赖腾讯云COS服务，大文件上传建议在Wi-Fi环境下进行

## 四、功能模块说明

### 4.1 对话与创作模块

#### 4.1.1 模块定位

对话与创作模块是软件的核心功能区，用户通过与AI进行多轮对话，获取内容创作建议、执行内容生成任务、分析上传素材，并进行创作全流程的交互式协作。

#### 4.1.2 页面构成

对话页整体采用上下分区布局：

- **顶部导航区**：显示页面标题"ClawSolo 对话"，左侧会话按钮用于展开/收起会话抽屉，右侧刷新按钮用于重新加载趋势数据
- **趋势数据看板区**：位于页面上半部分，包含：
  - 粒度切换器：支持"日""周""月"三种时间粒度切换
  - 更新时间显示：展示数据最后刷新时间
  - 导出报表按钮：支持将趋势数据导出
  - 四个统计卡片：今日新增粉丝、总播放量、总互动数、发布数量
  - 趋势折线图：以SVG形式绘制近7天（或对应周期）的趋势折线，展示访客、新增粉丝、播放量、互动数四项指标的变化趋势
  - 无数据提示：根据账号绑定状态显示不同提示（未绑定/数据同步中）
- **快捷问题区**：四个快捷问题卡片，点击后直接作为问题发送给AI
- **对话消息区**：展示当前会话的用户消息和AI回复，AI回复采用流式打字机动效逐字展示
- **输入操作区**：包含文本输入框、文件上传按钮、发送按钮

#### 4.1.3 主要操作

用户在对话页可执行以下操作：

- **发起AI对话**：在输入框中输入问题，点击发送按钮，AI将以流式打字效果逐字回复
- **使用快捷问题**：点击预设快捷问题卡片，快速发起常见询问
- **上传文件分析**：点击文件上传按钮，选择图片或视频文件，上传后文件将随问题一起发送给AI进行分析
- **查看趋势数据**：切换日/周/月粒度查看不同时间范围的趋势变化
- **导出趋势报表**：点击导出按钮，将当前趋势数据导出
- **管理会话**：点击左上角按钮展开会话抽屉，可新建会话、切换历史会话、删除会话
- **分享页面**：点击分享按钮，生成分享海报并分享到微信好友或朋友圈

#### 4.1.4 业务价值

对话与创作模块是创作者日常使用频率最高的功能区域。通过AI对话，创作者可以快速获得内容创意、标题建议、发布时间优化、粉丝画像分析等专业指导，大幅降低内容创作的决策成本。趋势数据看板则让创作者能够直观掌握账号近期表现，及时调整运营策略。

#### 4.1.5 关联关系

对话页与以下页面存在直接关联：

- 点击会话抽屉中的会话项，切换不同对话上下文
- 点击趋势看板的无数据提示中的"立即绑定账号"或"前往账号检测"按钮，跳转至账号检测页
- 通过分享功能将对话内容或页面分享至微信好友和朋友圈

### 4.7 账号检测与数据模块

#### 4.7.1 模块定位

账号检测与数据模块帮助用户绑定和管理自媒体平台账号，监测账号运行状态，查看趋势数据，并通过AI进行账号健康诊断。

#### 4.7.2 页面构成

账号检测页包含：

- **状态概览卡**：顶部紫色渐变卡片，展示已绑定账号数量和运行状态指示灯（绿色表示正常，黄色表示未绑定）
- **已绑定账号列表**：每个账号以卡片形式展示，包含平台图标、账号昵称、平台标签、绑定时间、运行状态、解绑按钮

## 五、典型业务使用路径

### 5.1 新用户首次使用路径

1. **登录**：打开微信小程序，点击"微信一键登录"授权，系统自动创建账号
2. **了解产品**：登录后浏览"功能介绍"页，了解平台能力和工作流
3. **体验AI对话**：进入"对话"页，选择快捷问题或输入自定义问题，与AI进行首次对话
4. **绑定自媒体账号**：进入"我的"→"账号检测"，选择平台并绑定自媒体账号
5. **查看趋势数据**：绑定成功后返回"对话"页，查看趋势数据看板（系统自动生成演示数据）
6. **购买套餐**：如需更多额度，进入"我的"→"套餐定价"选择套餐并完成微信支付

### 5.2 日常内容创作路径

1. **进入对话页**：从TabBar点击"ClawSolo 对话"
2. **查看数据趋势**：浏览顶部趋势看板，了解近期账号表现
3. **发起创作需求**：在输入框中描述创作需求（如"帮我写一条小红书爆款笔记"），或点击快捷问题
4. **接收AI回复**：等待AI以流式效果输出创作建议或内容
5. **上传素材分析**：如有需要，上传图片或视频让AI分析并给出优化建议
6. **保存对话结果**：对话内容自动保存至聊天历史，可在素材库中查看
7. **查看消耗记录**：在"我的"→"消耗记录"中查看本次服务的扣费明细

### 5.3 账号监测与优化路径

1. **进入账号检测页**：从"我的"→"账号检测"进入
2. **检查绑定状态**：查看已绑定账号列表的运行状态
3. **执行AI诊断**：点击"AI诊断"按钮，获取账号健康诊断报告
4. **查看诊断结果**：阅读AI生成的账号发展阶段判断、问题排查和优化建议
5. **返回对话页咨询**：根据诊断结果，在对话页进一步询问AI具体的优化执行方案
6. **查看趋势变化**：持续观察趋势数据看板中的指标变化，评估优化效果

### 5.4 套餐升级与充值路径

1. **进入定价页**：从"我的"→"套餐定价"进入
2. **对比套餐**：浏览6档套餐的额度与价格，选择适合的套餐
3. **发起购买**：点击"立即购买"，调起微信支付完成付款
4. **查看订单**：在"我的"→"我的订单"中查看购买记录和套餐生效状态
5. **额度不足时充值**：当套餐额度用尽时，进入"算力充值"页选择算力包充值
6. **追踪消耗**：在"消耗记录"中查看详细的用量和扣费情况

## 六、说明范围与边界

### 6.1 本文档说明范围

本文档说明范围限定于 Luna AI - ClawSolo 微信小程序前端可见的全部功能模块、页面构成、用户交互和业务逻辑。具体包括：

- 对话与创作功能
- 趋势数据看板展示
- 自媒体账号绑定与监测
- AI账号健康诊断
- 素材云存储管理
- 套餐计费与微信支付
- 消耗记录追踪
- 用户中心与系统设置
- 会话历史管理
- 公告通知系统

### 6.2 本文档未涉及范围

以下内容不在本文档说明范围内：

- 后端数据库架构设计、表结构和索引策略
- Edge Functions 内部实现细节和算法逻辑
- Hermes Agent 大语言模型的训练数据和技术原理
- Supabase 平台自身的运维和管理功能
- 腾讯云COS的底层存储机制
- 微信支付SDK的内部通信协议
- 第三方平台（小红书、抖音等）的API接口和数据规范
- 系统的部署拓扑、负载均衡和容灾方案

### 6.3 软件边界说明

- 软件为微信小程序形态，主要面向移动端用户，未提供PC端Web版本
- AI对话能力依赖外部Hermes Agent服务，当该服务不可用时对话功能将受限
- 趋势数据看板的数据来源依赖OpenClaw/Claw MCP的数据推送，未绑定或未推送数据时展示演示数据或同步中状态
- 微信支付功能仅在微信小程序正式环境中可用，开发环境和H5浏览器中无法使用
- 素材存储依赖腾讯云COS服务，该服务不可用时文件上传和预览功能受限
