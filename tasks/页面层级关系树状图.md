## 三、完整页面层级树（含导航缺陷标注）

```

│
├── login/                          【独立页面】登录/注册

├── features/       🧩【TabBar-功能】

│   └── ✅ 底部 TabBar 可随时回到其他根页面
│
├── profile/        👤【TabBar-我的】个人中心
│   ├── ⚠️ 头像编辑、昵称编辑（仅状态切换，无页面跳转）

│   ├── ❌ → navigateTo → pricing                           【定价】
│   │       └── 🚨【死胡同】页面无返回按钮，无法回到上一页或首页
│   ├── ❌ → navigateTo → materials                         【我的素材】
│   │       └── 🚨【死胡同】页面无返回按钮，无法回到上一页或首页
```

## 六、统一返回按钮规范建议

所有非 TabBar 子页面，建议统一在顶部添加返回导航：

```tsx
// 顶部导航栏示例
<div className="flex items-center gap-3 px-4 py-3">
  <div 
    className="w-10 h-10 rounded-xl bg-accent flex items-center justify-center"
    onClick={() => Taro.navigateBack()}
  >
    <div className="i-mdi-chevron-left text-foreground" style={{fontSize: '22px'}} />
  </div>
  <span className="text-2xl font-bold text-foreground">页面标题</span>
</div>
```

**需要添加返回按钮的页面清单：**
1. `src/pages/pricing/index.tsx`
2. `src/pages/monitor/index.tsx`
3. `src/pages/materials/index.tsx`
4. `src/pages/account-security/index.tsx`
5. `src/pages/settings/index.tsx`
6. `src/pages/about/index.tsx`

**需要修改导航方式的代码：**
- `src/pages/chat/index.tsx` 第 88 行：`navigateTo` → `switchTab`
