# Luna AI - ClawSolo 微信小程序 事实核查纪要

## 一、平台定位

Luna AI（ClawSolo）是一款面向自媒体创作者的AI内容生成与数据分析微信小程序，基于Taro框架开发，集成Hermes Agent（Luna 2.0）大模型能力，支持多平台内容创作、素材管理、趋势数据监测与账号健康分析。

## 三、各页面功能详情

### 3.1 对话页（chat）

- **AI对话区**：与Hermes Agent（Luna 2.0）进行多轮对话，支持文本输入和文件上传
- **快捷问题**：预设"如何提升内容播放量"、"分析我的粉丝画像"、"推荐爆款选题"、"优化发布时间"
- **趋势数据看板**：顶部展示近7天趋势数据折线图（新增粉丝、播放量、互动数、发布数量），支持日/周/月粒度切换
  - 统计卡片：今日新增粉丝、总播放量、总互动数、发布数量
  - 更新时间显示
  - 导出报表按钮
  - 空状态：未绑定账号时提示"未绑定自媒体账号"并引导跳转绑定；已绑定但无数据时提示"数据同步中"
- **文件上传**：支持选择图片/视频上传至Supabase Storage，上传后发送给AI分析
- **会话管理**：左侧抽屉展示历史会话列表，支持新建会话、删除会话
- **分享**：支持生成分享海报分享到微信好友和朋友圈
- **流式输出**：AI回复采用打字机动效逐字展示

### 3.2 功能介绍页（features）

- **Hero区**：品牌标语"Luna AI - 自媒体创作者AI工作站"，包含品牌色渐变背景
- **平台能力表格**：展示公众号、小红书、抖音、微博、飞书文档等平台的文案创作能力
- **AI图片生成示例**：三张示例图片轮播展示
- **典型工作流**：展示三个工作流卡片
  - 工作流1：抖音科普视频（3分钟脚本→6张配图→旁白+字幕→合成视频→质检输出）
  - 工作流2：小红书爆款笔记（备选标题→正文→封面图→发布建议→知识库）
  - 工作流3：公众号深度长文（搜索动态→大纲→3000字长文→飞书排版→存档）
- **差异化对比表**：视频生成、全流程自动、语音合成、多平台发布、本地知识库、定制化程度六项对比
- **技术底座层级**：AI小红书图、数据层、消息层、运行层四层架构展示
- **定价入口**：底部"立即开始"按钮跳转套餐定价

### 3.3 客服页（service）

- **LUNA介绍卡**：顶部紫色渐变卡片，展示LUNA头像和介绍
- **AI客服对话**：与LUNA进行客服对话，支持快捷回复（"查看我的套餐"、"如何使用AI创作"、"联系人工客服"）
- **快捷入口卡片**：客服回复后展示快捷跳转按钮
  - 查看我的套餐 → 定价页
  - 功能介绍 → 功能介绍页
  - 绑定自媒体账号 → 账号检测页

### 3.4 我的（profile）

- **用户信息卡**：头像（可点击更换）、昵称（可编辑）、套餐等级标签、会员有效期
- **公告通知**：顶部公告轮播，支持展开/收起，类型包括新品上线、版本更新、系统维护、通知
- **快捷入口区**：4个入口卡片
  - 我的订单 → 订单页
  - 素材库 → 素材库页
  - 消耗记录 → 消耗记录页
  - 账号检测 → 账号检测页
- **设置列表**：
  - 账号安全 → 账号安全页（修改密码、退出登录）
  - 系统设置 → 系统设置页（通知、缓存、个性化）
  - 关于我们 → 关于页（服务协议、版本信息）

### 3.5 登录页（login）

- **微信一键登录**：调用微信登录获取openid，自动创建/关联账号
- **用户名密码登录**：支持已有账号密码登录
- **隐私政策链接**

### 3.6 套餐定价页（pricing）

- **套餐对比区**：6档套餐卡片横向排列
  - 免费版：0元/月，3条图文/月
  - 图文版：9.9元/月，30条图文/月
  - 视频入门版：29.9元/月，15条图文+60秒视频/月
  - 视频专业版：99元/月，60条图文+300秒视频/月
  - 专业版：299元/月，200条图文+1200秒视频/月
  - 企业版：999元/月，不限图文+不限视频
- **当前套餐高亮**：已购套餐显示"当前套餐"标记
- **算力充值入口**：底部"单独充值算力"入口
- **购买按钮**：选择套餐后点击"立即购买"，调起微信支付
- **订单查询**：底部"查看我的订单"入口

### 3.7 订单页（orders）

- **当前套餐卡片**：紫色渐变卡片展示当前套餐等级、有效期、图文/视频额度
- **余额信息卡**：创作余额（人民币）
- **算力充值入口**：点击跳转算力充值页
- **订单列表**：展示历史购买订单，含订单状态（待支付/已支付/已取消/已退款）、金额、时间
- **底部导航**：升级套餐按钮 + 消耗记录按钮

### 3.8 素材库页（materials）

- **Tab切换**：COS云存储 / 聊天历史
- **COS文件区**：
  - 分类筛选：全部/图片/视频/音频/文档/数据/其他
  - 类型筛选：全部/上传文件/AI输出
  - 文件列表：卡片式展示，含文件名、大小、时间、类型图标
  - 上传按钮：选择本地文件上传至COS
  - 预览：点击图片/视频预览
  - 删除：长按或点击删除
- **聊天历史区**：
  - 会话列表展开/收起
  - 展示每轮对话的消息记录

### 3.9 账号检测页（monitor）

- **状态概览卡**：顶部紫色渐变卡片，展示已绑定账号数量、运行状态
- **已绑定账号列表**：卡片式展示每个绑定的自媒体账号
  - 平台图标（小红书/抖音/B站/微博/微信/快手/其他）
  - 账号昵称、平台标签、绑定时间
  - 运行状态"运行正常"
  - 解绑按钮
- **绑定新账号**：展开表单选择平台、输入账号昵称和ID
- **AI账号诊断**：点击"AI诊断"，调用Hermes Agent生成账号健康诊断报告
- **数据同步提示**：OpenClaw/Claw MCP 数据同步说明
- **初始化看板**：点击"初始化看板演示数据"生成近7天演示趋势数据

### 3.10 消耗记录页（usage-records）

- **消耗记录列表**：展示算力消耗明细
  - 时间、类型（图文/视频/对话）、用量、算力消耗值
- **底部快捷按钮**：去充值、查看订单

### 3.11 系统设置页（settings）

- **通知设置**：消息推送开关
- **缓存清理**：清理本地缓存按钮
- **个性化设置**：主题/字号等设置项

### 3.12 账号安全页（account-security）

- **修改密码**：原密码、新密码、确认新密码输入框
- **退出登录**：确认退出按钮

### 3.13 关于页（about）

- **服务协议**：点击展开查看
- **隐私政策**：点击展开查看
- **版本信息**：当前版本号
- **联系我们**：客服联系方式

### 3.14 算力充值页（compute-recharge）

- **算力包选择**：多个算力包卡片，含算力值和价格
- **微信支付**：选择后调起微信支付
- **充值记录**：展示历史充值记录

### 3.15 财务管理页（admin-finance）

- **收入统计**：今日/本周/本月收入
- **订单管理**：所有用户订单列表
- **退款处理**：退款申请审核

## 四、交互验证

### 页面跳转关系

```
TabBar: 对话 ←→ 功能介绍 ←→ 客服 ←→ 我的
                  ↓             ↓         ↓
               定价页         快捷入口   订单/素材/消耗/监测/安全/设置/关于
               ↕                         ↕
             订单页  ←────────────────→  消耗记录页
               ↕                         ↕
            充值算力 ←──────────────────→
```

### 关键交互

- **登录**：微信一键登录自动创建账号，或用户名密码登录
- **AI对话**：输入消息→发送→流式输出回复，支持文件上传和图片分析
- **购买套餐**：选择套餐→微信支付→订单状态更新→套餐生效
- **绑定账号**：选择平台→输入信息→确认绑定→自动生成演示数据
- **数据看板**：粒度切换（日/周/月）→刷新数据→导出报表
- **素材上传**：选择文件→上传至COS→文件列表刷新
- **头像更换**：点击头像→选择图片→上传至Storage→更新profile

## 五、数据与存储

- **Supabase**：Auth认证、PostgreSQL数据库、Edge Functions、Storage存储
- **COS（腾讯云对象存储）**：素材文件长期存储
- **微信支付**：套餐购买和算力充值
- **Hermes Agent**：AI对话和内容生成（152.136.47.2:8642）

## 六、技术栈

- 前端：Taro 3.x + React + TypeScript + Tailwind CSS（UnoCSS）
- 后端：Supabase（PostgreSQL + Edge Functions）
- AI：Hermes Agent（Luna 2.0，OpenAI兼容接口）
- 存储：Supabase Storage + 腾讯云COS
- 支付：微信支付
