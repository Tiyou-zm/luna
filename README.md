# Luna Miniapp

Taro + React + TypeScript WeChat Mini Program project for Luna material generation workflows.

## Structure

- `src/app.config.ts`: Mini Program routes and tab bar.
- `src/client/supabase.ts`: Supabase client and Taro request adapter.
- `src/pages/`: Mini Program pages.
- `src/utils/cos.ts`: Tencent COS upload helper.
- `supabase/functions/`: Edge Functions for auth, Hermes, COS credentials, and material workflows.
- `docs/dev-progress.md`: Current development and verification log.

## Development

```bash
pnpm install
pnpm run dev:weapp
```

WeChat DevTools should open the repository root. `project.config.json` points `miniprogramRoot` to `dist/`, which is generated by the Taro build.

## Verification

```bash
pnpm run build:weapp
pnpm exec tsgo -p tsconfig.check.json
```
